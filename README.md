# Projeto React Native - Lista de Contatos

Este projeto foi desenvolvido como atividade prática utilizando React Native.

O objetivo é criar cinco telas com navegação entre elas, simulando um sistema de login e gerenciamento de contatos. Nesta etapa não há consumo de API externa, os dados são armazenados em arrays locais.

## Tecnologias Utilizadas

* React Native
* Expo
* React Navigation (native-stack)
* React Native Components (View, Text, TextInput, TouchableOpacity, FlatList)

## Instalação

Clone o repositório:

```
git clone https://github.com/seu-usuario/Atividade1-Mobile-Pages.git
```

Entre na pasta do projeto:

```
cd nome-do-projeto
```

Instale as dependências:

```
npm install
```

## Executar o Projeto

```
npx expo start --tunnel
```

## Estrutura de Telas

O projeto contém as seguintes telas:

* Login
* Cadastro de Usuário
* Lista de Contatos
* Cadastro de Contato
* Edição de Contato

## Funcionalidades

* Navegação entre telas utilizando Stack Navigator
* Lista de contatos utilizando FlatList
* Dados simulados em array
* Layout simples com estilização em tons de rosa

## Observações

* Nesta versão o projeto não consome API externa.
* Os dados não são persistidos.
* A integração com backend será implementada em atividade futura.
